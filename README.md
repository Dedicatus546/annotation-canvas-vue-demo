# A demo that show how to create a annotation on a canvas element

## start

install deps.

```bash
pnpm run build
```

start dev server

```bash
pnpm run build
```

open http://localhost:5173 in your browser.

then you can see follow page. and you can drag you mouse in rect to create a annotation.
